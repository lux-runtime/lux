### âš¡ What is Lux?

**Lux** is a modern, standalone runtime for **Luau** (the language used by Roblox), designed to compete directly with Python for scripting and automation.

It extends the safety and speed of Luau with a powerful **Foreign Function Interface (FFI)**, allowing you to use C/C++ libraries directly in your scripts without glue code.

### ðŸš€ Key Features

* **Native FFI:** Load `.dll` / `.so` files and call C functions directly from Luau.
* **LPM (Lux Package Manager):** A `npm`-style package manager with support for pre-built binaries.
* **Blazing Fast:** Built on top of the highly optimized Luau VM and Rust.
* **Battery Included:** Comes with File System, Networking, and Process modules out of the box.
